## 📝 核心代码片段

### 扩散噪声添加（最重要）
```python
def add_noise(self, actions, grip_actions, timesteps):
    # 1. 转换为6D (tx, ty, tz, rx, ry, rz)
    actions_6d = transforms_to_actions(actions)
    
    # 2. 归一化到[-1, 1]
    actions_6d = self.normalizer.normalize_actions(actions_6d)
    
    # 3. 添加噪声（核心！）
    noise = torch.randn_like(actions_6d)
    noisy = self.noise_scheduler.add_noise(actions_6d, noise, timesteps)
    noisy = torch.clamp(noisy, -1, 1)
    
    # 4. 反归一化
    noisy = self.normalizer.denormalize_actions(noisy)
    
    # 5. 转回4x4矩阵
    noisy = actions_to_transforms(noisy)
    
    return noisy
```

def subsample_traj(traj, grips, trans_space=0.01, rot_space=3):
    result = [traj[0]]
    for i in range(1, len(traj)):
        trans_dist = norm(traj[i][:3,3] - result[-1][:3,3])

## ❓ FAQ

**Q1: 哪个策略最重要？**  
A: **扩散噪声**（策略7）是核心，其次是轨迹子采样和演示排列组合。

**Q2: 如何快速增加数据量？**  
A: 通过演示排列组合，从N个演示可生成C(N,2)个样本对。

**Q3: 扩散噪声如何工作？**  
A: 在训练时随机采样时间步t，向真实动作添加噪声，模型学习去噪。

**Q4: 为什么需要归一化？**  
A: 扩散过程需要数据在标准范围内，通常是[-1,1]。

**Q5: 可以禁用某些增强吗？**  
A: 可以，通过config设置。但扩散噪声是核心，不建议禁用。
