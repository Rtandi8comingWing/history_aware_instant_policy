物体附加工作流：
  
  [1] 夹爪关闭
      ↓
  [2] 查找最近物体（遍历所有物体，计算距离）
      ↓
  [3] 距离 < 10cm？
      ├─ Yes → 附加物体
      │         - 保存 attached_object
      │         - 计算 attachment_offset（在夹爪坐标系）
      └─ No  → 不附加
      ↓
  [4] 夹爪移动
      ↓
  [5] 有附加物体？
      ├─ Yes → 更新物体姿态
      │         new_pose = gripper_pose @ attachment_offset
      │         scene.set_pose(obj, new_pose)
      └─ No  → 不操作
      ↓
  [6] 夹爪打开
      ↓
  [7] 分离物体
      - attached_object = None
      - attachment_offset = None
